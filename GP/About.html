Status: Makeing a Base in Rust

What is it?

A simple and basic GUI and TUI set for Gentoo's Portage package manager made by SP649/lovemearowlet for LearnixOS. This is just a hobby project so it will not be fastly developed!

Why should I use it?

It is a super basic and easy way to install apps for Gentoo. It is made for anyone that wants it of any level or skill.
